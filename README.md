# Motivational To-Do App — Overview

A lightweight To-Do list web app built with plain **HTML**, **CSS**, and **vanilla JavaScript** that helps you add, edit, reorder, and complete tasks with delightful, motivating feedback.
Link -  https://ishwari-kenchi.github.io/ToDo-List/

**Overview**

- Add, edit (double-click), and delete tasks.
- Mark tasks as completed with an animated check and confetti celebration.
- See progress with a smooth percentage progress bar and a celebratory banner when all tasks are done.
- Reorder tasks using drag-and-drop (desktop and touch).
- Export and import tasks as JSON for backups.
- All tasks persist locally in `localStorage` so they stay across browser refreshes.
- Responsive and accessible: works on mobile and desktop, with keyboard-friendly interactions.

Built to create small, rewarding moments—finish a task, get a little celebration, and feel motivated to do the next one.
